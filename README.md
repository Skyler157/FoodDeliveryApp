# FoodDeliveryApp

A full-stack food delivery website built with React JS, MongoDB, Express, Node.js, and Stripe for a seamless and secure online food ordering experience. This project allows users to browse menus, add items to their cart, place orders, and make payments via Stripe, while also offering admin functionality to manage orders and restaurants.

Tech Stack
Frontend: React JS
Backend: Node.js with Express
Database: MongoDB
Payment Gateway: Stripe API
Authentication: JWT (JSON Web Token) for secure authentication

Features
User Features:
View restaurant menus
Add items to cart
Place and track orders
Secure payments through Stripe
User login and registration
Admin Features:

Restaurant owner dashboard
Manage menu items
View and manage orders
Security:

JWT Authentication for secure login
Password hashing for user safety
